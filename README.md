# TabDSP Feedback

Bug reports, feature requests, and feedback for **TabDSP** — a per-tab professional audio DSP extension for Firefox.

## What is TabDSP?

TabDSP brings broadcast-quality audio processing directly to your browser:

- **8-band Parametric EQ** with real-time spectrum analyzer
- **Single-band & Multiband Compression** with gain reduction metering
- **True Peak Limiter** with ITU-R BS.1770-4 compliance presets
- **Professional Metering** (Peak, RMS, LUFS)
- **Linear Phase EQ** and **M/S Processing** modes
- **Drag-and-drop module reordering**

## Reporting Issues

Please [open an issue](https://github.com/JohnKessler/tabdsp-feedback/issues/new) and include:

1. **Firefox version** (check `about:support`)
2. **TabDSP version** (check `about:addons`)
3. **Website** where the issue occurs
4. **Steps to reproduce** the problem
5. **Expected vs actual behavior**
6. **Console log** if available (F12 → Console tab)

## Feature Requests

Feature requests are welcome! [Open an issue](https://github.com/JohnKessler/tabdsp-feedback/issues/new) with the `enhancement` label and describe what you'd like to see.

## Status

TabDSP is currently in **pre-beta**. Expect rough edges. Your feedback helps shape the product.
